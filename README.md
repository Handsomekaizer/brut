# Script_GenerateMnem_Check_Balance
A script to generate random 12 words mnemonic and checking balances.

**How to use it:**

- Add your Infura key: ```const web3 = new Web3('https://mainnet.infura.io/v3/**Infura Key**');```
- Run it.
